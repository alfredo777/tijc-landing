# Torneo Internacional Jorge Campos 2026

Landing page para el Torneo Internacional Jorge Campos 2026.

## Instalación

```bash
cd torneo-jorge-campos
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre http://localhost:3000

## Producción

```bash
npm start
```

## Características

- ✅ Express.js + Handlebars
- ✅ Hot Reload automático
- ✅ Galería con Lightbox
- ✅ Filtros de galería
- ✅ Countdown
- ✅ Modales de información
- ✅ Responsive design
- ✅ Tailwind CSS

## Hot Reload

- Los cambios en CSS se aplican sin recargar
- Los cambios en imágenes se actualizan automáticamente
- Presiona **R** en el navegador para recargar manualmente# tijc-landing
